▌志工  ▌


// 推主燈者 //

需要15人至20人輪流協力推動主燈車，包括2至3人拉動連燈串。

門神作為民間文化常見的視覺圖像，此次燈車以「開」與「關」的象徵重新詮釋門神，呼應上元節「萬戶開鎖」的意象，同時也是節祭開啟的非日常空間的守護。藉此，希望對交界處的「門」產生不同的感受可能。

側邊柱則以花磚裝飾。在台灣文化當中，花磚象徵一種文化交流，以不粗暴的方式，重新妝點居所，期望節祭也能找到一種文化交流的方式，建立在彼此都有足夠的自信，能互相欣賞而非相互爭鬥。


// 推鼓車者 //

需要2至4人輪替推動鼓車。

鼓車透過音樂創造場域，鼓車所在之處便是慶典之所在。
帶著鼓車，一起創造移動的祭典。


// 後台機動群 //

成為上元後盾，機動協助現場事物。

－交管：引導現場，指揮交通。
－補給：飲水補充，醫療協助。
－宣傳：發放酷卡，介紹活動。
－万事屋：跑東跑西，交辦支援。


// 動靜態攝影 //

成為眼睛，紀錄現場。
靜態2人，動態2至3人。

（備註：惟登記攝影師得取得識別布條，未登記者需配合現場指引動線進行紀錄。）



▌參與者  ▌

// 自提燈籠上街 //

只要為我們留下放蠟燭點火的空間，
無論是古早味的奶粉罐，還是切半的鐵罐，
讓我們路上相見！


// 自帶樂器上街 //

鈴鼓也好，直笛也罷，
帶上你的樂器，一起上街即興彈奏，放夜歌舞。
