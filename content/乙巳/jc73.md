# 關於歌名

這首歌有個響亮的名字——「慶津梁」。這個詞，大家可能不太熟悉，但「津梁」指的是「渡口上的橋樑」，這是個大家可能都看過，但不一定知道名字的地方。簡單來說，就是幫助船隻在渡口停靠的橋樑，尤其是在水位較低、泥沙淤積的地方，津梁可以讓船隻安全地停靠。

我最開始對「津梁」感興趣，是因在沖繩看到這個詞被非常認真地使用。沖繩是座小島，一直是東、西方航海的交匯處。當地有口大鐘，刻著「萬國津梁」四個字，象徵著沖繩作為航海路徑中的一個重要「渡口」。當時聽到，覺得很有趣，一個地方居然樂於將自己作為渡口？

但其實高雄、甚至台灣的歷史，也很跟海、港緊緊相依，許多文化、故事在這裡交織。尤其是高雄這座工業大城，一直是移民聚集地，匯聚了來自各地的力量。

我感覺，高雄的港口也像是一個「渡口」，不只是外界進入台灣的地方，更是台灣文化與世界接軌的一個窗口。這讓我覺得，台灣可以像沖繩一樣，擁有一個「渡口」的身分，並且擁有這樣的身分也應該是值得慶祝的。因此，我們就將歌曲命名為《慶津梁》。

或許《慶大港》也是個有意思的名字，但津梁更為隱晦，不會讓人一聽到歌曲就馬上聯想到「高雄港」。所以，這首歌不會被侷限在某一個地方。「津梁」可以更有象徵性，讓人能對這個詞擁有更多的想像和認識。

這首歌的結構也特別，我們總共用四段不同的故事，來表達不同的主體和情感，不會是太單一的敘事方式。

> 清明端午過元宵，同齊點燈閣一擺

> 
> 十五暝，慶津梁，共許一個百年久長

# 第一段解析

在《慶津梁》的第一段，我們講了高雄作為移民城市的特質。最開始的「移民城市人來去，家譜三代三所在」，就反映高雄的歷史和特點——一個多元且不斷變化的城市，很多人並不是在這裡出生，而是從其他地方遷來的。

我們之前在清明活動中，曾經一起討論自己的族譜，很多人的祖父母來自雲林西螺、嘉義朴子、台南學甲之類的地方，而來自高雄本地的家族卻少之又少，甚至連三代、四代都是高雄人的也非常罕見。這讓我們發現，高雄這座城市，是由各地移民所組成的。

移民城市有個特性，就是東西變遷得很快，很多東西是留不住的。現在的年輕人對老房子的記憶逐漸模糊，因為許多老建築在一代的時間就會消失。

我自己成長過程中，很多店家、商店，也在城市發展中無可避免地消失，讓人有點悵然。這是我這一輩的情形，而我們做田野的時候，發現文化中心這一代，在五十年前，甚至都還是農田，對老一輩的爺爺奶奶而言，這些變遷無疑是更為劇烈的。這背後當然有一種無力感，但我們這些百姓不過是是做藝術的平民，沒辦法真正停止這樣的變遷，只能用紙糊保留下一點當時建築的痕跡。

這樣一來，三個世代對這座城市有著完全不同記憶的情況下，世代之前很容易爭執誰的時代更好，但我在思考，在這樣的變遷之下，有什麼是帶得走的，然後我們找到的東西是「節慶」。所以歌詞中就提到，「清明端午過元宵，同齊點燈閣一擺」，雖然是不同世代，但我們都有那麼一點，對於過節的記憶。

也因此，就算看見這些變遷，我們能不能有個比起惆悵，更為正面的方式？我們不想要悲傷，而是再來點一次燈，不必擔心曾經經歷過什麼，不管是苦難還是歡樂，我們都可以一起重新出發。任何時候都可以開始。

所以，「莫驚生份到海墘，文火愈濟滿天星」代表的是一種慢慢積聚力量的過程。大家住在城市裡，鄰里之間很生份，但正如同點燈的過程一樣，慢慢地，眾多的微弱光芒會在海邊匯聚，形成一片燦爛的星空。這象徵著儘管我們是微小的，但如果集結力量，相約過節，最終就能形成滿天星。

而副歌這邊，「梳妝打扮地基主、厝邊隔壁來寫新族譜」，是想要來針對傳統玩味一下。因此，我們的副歌寫到：「梳妝打扮地基主，厝邊隔壁來寫祖譜。」這可能讓大家有點疑惑，因為地基主並沒有一個具象的形象，它是一個比較抽象的概念。但這次我們用了文學的手法，試著把地基主形象化，視為一個真實存在的象徵——它可能是什麼？我們也說不準。但我們帶著一種新的想法來看待它。

在過去，長輩們祭拜地基主的時候，可能比較嚴肅，但如果地基主能夠象徵我們對於新家的期待，那麼有沒有可能，我們也可以為它做些裝飾呢？就像家茗剛剛說的，每個剪紙上的人物都有一朵花，那我們是不是也可以給地基主一朵花呢？其實，地基主的存在，不只是傳統信仰，而是代表我們對新家的想像與期盼。

接下來一句「厝邊隔壁來寫新族譜」，其實也反映了移民城市的特性。在這樣的環境裡，身邊的鄰居往往彼此不認識，再加上小家庭化，大家對於族譜、家族的概念越來越淡薄。但這句話帶有一種祝福的意味——有沒有可能，當我們遷移到新環境時，對新家有新的期望，對鄰居有新的想像？也許我們不必拘泥於過去的血緣關係，而是能夠在新的地方，與新的夥伴一起建立新的連結。

所以才會寫到「不要管時代差了天地」，因為我們根本無法完全理解阿公阿媽那一輩所經歷的事情。時代變遷太快了，當我們進行田野調查時，常常覺得過去的一切恍如隔世。例如，一座大樓是怎麼蓋起來的？一座港口是怎麼發展成現在的樣子？短短一百年，世界已經發生了如此巨大的變化，甚至讓人覺得像是跨越了三百年。

但這些變遷，畢竟都是同一代人的記憶。我們能否讓那些歷史的傷痕、時代的隔閡稍微淡化一些？透過元宵節這樣的傳統節日，大家或許能夠找到一些共同的記憶，讓彼此的故事重新連結在一起。

# 第二段解析

在《慶津梁》的第二段，我們從高雄的歷史，看到過去華僑移民的歷史。特別是福建、廣東的漢人移民下南洋到東南亞的歷史。這段歷史對當代人來說可能較為陌生，其實這些移民之所以出海，是因為當時福建的土地貧瘠，山海相間缺乏足夠的耕地，很多人不得不走上遠航的路。

這群漢人不僅來到台灣定居，也有大量的移民前往馬尼拉、檳城等地，這些地方成為了漢人社群的集聚點。而金門便因為獨特的地理位置，成為華僑故鄉，成為了當時許多漢人出海的起點，金門的莿桐也因此落南洋。

雖然今天我們不再乘船遠行，而是搭乘飛機，但移民的情感卻是相通的——我們依然面臨著離開家鄉、追求更好生活的選擇，而這種情感的共鳴，並不會因時代的變遷而消失。

當時的漢人移民下南洋打拚，歌詞中提到的「批信」或「僑批」，便是當時移民們用來與家鄉保持聯繫的方式。當時的華僑尤其重視郵政系統，因為這是他們與家人聯繫、寄送錢財的重要途徑。信件成為了他們與家鄉的唯一聯繫，並成為了他們生活的一部分。這段歌詞中「批信講盡海上事，銀票心肝寄轉厝」也表達了那個年代，漢人移民們對家鄉的思念與對家人的責任感。

除了離別，下南洋為也為閩南建築帶來新的建築風格，特別是「花磚」裝飾。我在新加坡、馬來西亞、泉州、台灣洋樓，都能看過，這些花磚對當時的漢人而言，成為下南洋的一個象徵，他們將當地的殖民地建築風格帶回故鄉，創造出了獨特的南洋風建築。而因為港口，人來去，生疫病，導致王爺送瘟神信仰，也成為這群以海為生的漢人，重要的信仰。

講到這邊可以發現，當我們把視角放到這些漢人移民上的時候，我們文化的可能性變得非常不一樣。所以，我們在第二段的歌詞加入了「王爺保庇」與「番仔樓貼花磚」，進一步傳唱這些漢人移民的故事。

最後，副歌部分的「十五暝，慶津梁」再次回到歌名的主題，這次則是想要去描繪，漢人移民在海路上結交的四海朋友，也能去享受這樣的佳節，大家仍然可以在這個節慶的時刻共同慶祝，無分彼此。

同時，也提到「莫在佳節來分開」，這件事情真的很真實。除夕那天，我剛好在外面，因為要畫門神燈繪，所以沒有吃年夜飯。結果發現，還有一大堆人在工作，讓我覺得不可思議。或許，很多人真的沒辦法回家過年吧？當下心裡有一種很複雜的感受——一方面慶幸有人還在工作，我才有便當可以吃；但另一方面，也覺得有點哀傷，因為這代表著，真的有許多人今年是回不了家的，他們必須待在外地工作。

如果把這種感受再放大來看，其實這樣的離別情境一直都存在。以前的華僑在出行時，必須順應季風和洋流，只有在對的時間才能回家。他們沒辦法說「我今天有急事，要馬上回家」，因為這在當時是完全不可能的事。

所以，我們最後想要強調的，是要珍惜今天的便利。如今，交通發達，回家變得相對容易，但在這個同時，我們也應該試著去記得——分開的心情，對所有人來說，始終都是難受的。

# 第三段解析

我們這次的討論從移民談到華僑，最後回到了台灣。我們選擇從台灣的漢人歷史出發，雖然其中涉及原住民的部分，但我們希望先從自身熟悉的脈絡開始。

談到文化的延續與傳承，不得不提到陳達的《思想起》，這首歌承載了台灣移民的歷史與情感。他的嗓音滄桑而深遠，讓人彷彿能夠聽見先民渡海時的無助與艱辛，乃至於台灣這四百年來的風風浪浪。

另一方面，林懷民的《薪傳》則以舞蹈展現了這段歷史。在1978年中美斷交的夜晚，《薪傳》的演出激起了全場的共鳴，成為台灣人共同的記憶。我們這一代雖然沒經歷那個時代，但這些作品已經成為有歷史感的文化遺產，見證了台灣不同時期的變遷。

我們不希望只是透過文字去理解歷史，而是希望歷史能夠「有聲音」。因此，我們在寫作時特別選用了《思想起》，希望讓讀者在閱讀時，腦海中浮現陳達歌聲的畫面，那種滄桑與厚重，能讓人感受到移民過程中的掙扎與堅韌。

不過，台灣的歷史不僅是戰爭與鬥爭的故事，更是一個融合與創造的過程。例如，台灣的許多植物其實是外來的，例如荷蘭人帶來的木棉花與阿勃勒。阿勃勒最初甚至是荷蘭人為了軍事用途而種植，然而，數百年過去，人們已經遺忘了這層歷史，留下的只是美麗的花樹。

同樣地，日本統治時期留下的大量建築，今日已成為台灣城市美學的一部分。這些建築與台灣的文化融合，成為台灣人日常生活的一部分，讓我們反思，如何用新的方式去認識這些過去的遺產。

融合不只體現在建築與植物上，也體現在語言、音樂與飲食。例如，客家的織繡與原住民的十字繡技法在六堆地區交融，創造出獨特的紋樣；恆春的《守牛調》雖然是漢人歌曲，卻融合了排灣族的音調與尾音「咧」，形成了獨特的台灣音樂風格。

回到日常生活，飲食更是融合的最好例子。客家粿仔、原住民的馬告料理、從南洋傳來的沙茶醬、甚至如今已被視為台灣代表食物的牛肉麵，都是不同族群文化交會的產物。我們早已在日常生活中接受了這些融合，但或許我們很少停下來思考，這些食物背後的故事。

我們這一代成長於一個充滿紛爭的時代，媒體報導中充斥著衝突與對立，但事實上，融合的過程一直在發生，並且塑造了我們今天的台灣。我們的祖先無論來自何方，最後都在這片土地上安身立命。有名台灣作家林鼎鈞曾經寫道，**「每個祖先都是流浪者，他們流浪到的終點，就是家。」**

我自己曾經在路邊看過一場野台戲，當時我注意到台下的觀眾，有幾位大叔，面相看起來很兇悍，卻目不轉睛地盯著戲台，神情投入得像個孩子。他們的臉上沒有了社會賦予的威嚴，而是專注於戲曲情節，甚至有人眼中泛著光。這種景象讓我印象深刻，在野台戲的現場，無論是工人、農民、商人還是知識份子，大家都聚在一起，專注地看著台上的演出。那一刻，人們沒有階級之分，只有純粹的感動與共鳴。

所以，歌詞最後，我們埋了一個小巧思，過去台灣的元宵節有「鑽燈腳」的習俗，祈求添丁，象徵延續生命與希望。在今天的社會，或許生育不再是唯一的選擇，但我們仍然希望能在這片土地上，看見未來的可能性。

我們的世界充滿了爭執，人們為了水、土地、權利而爭鬥，但也有人選擇彼此融合，結為親戚，交換技術與文化，讓這塊土地變得更豐富。我們希望，在這個充滿對立的時代，能夠提醒彼此，台灣的歷史不只是衝突，而是融合與創造的歷程。

# 第四段解析

我們最後一段的重點是回到高雄。或許這和第一段有些相似，但我們關注的角度很不一樣。第一段是關於我們個人如何移民到高雄，而這一段，我們要寫的是高雄本身。

從歷史來看，高雄一直是台灣重要的產業城市。香蕉、鳳梨這些農產品，曾經是高雄的象徵。過去大家提到「香蕉碼頭」，那可是台灣對外貿易的重要據點。高雄在不同時代，承載著不同的經濟使命。曾經，高雄因為糖業興盛，而被稱為「糖港」。這是一個很多人不熟悉的過去，但它確實存在過，充滿甜味的記憶。

之後，隨著時代變遷，香蕉、鳳梨加工業發展，大樹地區的鳳梨加工廠成為重要的歷史建築。隨著工業的進步，高雄經歷了輕工業、重工業的轉變。從前鎮漁港、旗津沿海到楠梓工業區，再到今天高科技園區的建立，高雄的產業變化，反映出台灣經濟結構的轉型。

然而，高雄港曾經是個封閉的地方，禁止民眾進入，這樣的狀態一直持續到二十幾年前才改變。這使得高雄與基隆港的氛圍截然不同。基隆港是熱鬧的，有著滿滿的市民記憶；但高雄港，曾經是一個陌生而遙遠的存在，一個專注於「大事」的地方。

對我們這一代來說，我們的童年經歷了高雄港開放的過程。我們曾經無法靠近這片海，對港口的記憶和上一代截然不同。這樣的變遷，讓我們反思，高雄的發展與轉變，並非單一的工業成長，而是夾雜著社會與人們生活方式的改變。

回到這座城市的樣貌，高雄的工業發展帶來了繁榮，也帶來了污染。我們畫高雄風景時，發現不管朝哪個方向望去，工廠與加工區無所不在。往南，林園加工區的鋼鐵圍籬清晰可見；往北，高鐵沿線經過楠梓的工廠群；往西，大寮的工廠密集排列；甚至在港邊，造船廠與貨運碼頭依舊繁忙。從飛機上俯瞰高雄，這座城市被無數幾何形狀的工業建築包圍，宛如一座被鐵壁環繞的城堡。

我們這一代人，成長於高雄的文化中心一帶，或許遠離了這些工業區的日常勞動。但這些工業從未離開我們的生活。我們的父母或許穿著西裝上班，與工業生產保持距離，但工業依舊是這座城市的脈絡。我們對它的理解不深，也無法立即理解，但我們知道，它始終存在。

這使得我們在思考「元宵節」的意義時，陷入了長久的掙扎。我們希望這個節慶能夠超越階級，讓不同生活背景的人都能參與。但要如何讓高雄的工業勞動者也能共襄盛舉？我們發現，我們對他們的生活幾乎一無所知。

高雄不只是工業城市，它也是台灣環境運動的重要發源地之一。在發展工業的同時，人們開始思考保護環境的重要性。因此，衛武營、中央公園、柴山、愛河得以被保存下來。甚至在某個時期，愛河差點被填平，變成快速道路——這讓我們難以想像。但這一切都說明了，在工業化的高雄，人們不僅在建造工廠，也在努力保護城市的綠地。

不同階級的人對生活的需求不同，彼此的視角也不同。工人或許覺得我們「附庸風雅」，而我們則可能覺得他們「粗俗」。但其實，無論是士農工商，每個人都在為了生活努力。這座城市的發展，不是單純的工業化，而是在工業與自然、公園與工廠之間，找到一種共存的方式。

回到高雄的記憶，這座城市的聲音，始終帶著鐵的質感。小時候，我們搭渡輪前往旗津，那鋼製階梯發出的「嘰嘰」聲，至今仍清晰可聞。二十年後，再次搭上渡輪，那熟悉的機油味，依然留在空氣中。

高雄是一座「硬」的城市，它的街道筆直、工業井然有序。從一心路到十全路，不容易迷路，因為它的規劃是方正的。這座城市的產業歷史帶來了堅實的基礎，但同時，它也孕育著柔軟的生命。

無論環境多麼嚴峻，柔軟的事物依然存在。小孩會長大，海潮依舊拍打著西子灣，木棉花仍然會開放，月光還是會照在這片鋼鐵之城上。

今天，我們站在這裡，見證著時代的變遷。港口開放了，環境運動保護了許多事物，而我們，作為這個城市的新一代，能做些什麼？答案或許很簡單——我們還是繼續做我們的元宵節吧。有人唱歌，有人跳舞，有人擊鼓，有人吹奏。高雄的四季變化不大，無論是夏日的酷熱還是冬天的微涼，元宵節一年年地過去。

最後，我們還是回到同樣的一句話——「共許一個百年長久。」
